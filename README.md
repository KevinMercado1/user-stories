# Product and Annotation Manager

A simple tool designed for end-users to register products, manage annotations, and track financial subtotals for each item.

---

## Overview

This application provides a straightforward way for clients to maintain an organized list of products. It focuses on two core functions:

1. Data Entry: Registering new products along with specific annotations.
2. Financial Tracking: Calculating subtotals to show the total money collected per product.

## Features

- Product Management: Add and list new products.
- Annotations: Input custom notes for every item.
- Subtotal Calculation: Automatic calculation based on unit price and quantity.
- Revenue Reports: Displays the total amount earned for each specific product entry.

## Logic

The program calculates the financial data as follows:

Subtotal = Unit Price \* Quantity
